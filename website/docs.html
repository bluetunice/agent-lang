AFL 语言文档

AgentFlow Language (AFL) v1.6 — 专为 Agent 开发设计的领域特定语言。

v1.6 更新: 新增 425 个 Python 单元测试，全面覆盖词法分析、语法解析、AST 节点、解释器及 CLI 全链路。同时修复了词法分析列号计数、逻辑运算符转换、模块导入查找、高阶函数传参、正则替换参数顺序等 6 项 bug。运行 python3 -m pytest tests/ -v 查看完整测试报告。

安装与运行

AFL 解释器使用纯 Python 实现，无第三方依赖。

终端
# 运行 .agent 文件
python3 afl_lang/agent.py examples/hello.agent

# REPL 交互模式
python3 afl_lang/agent.py repl

# Python API
from afl_lang import run_agent, run_file

run_agent('print("Hello, World!")')
run_file("examples/hello.agent")

基础语法

AFL 采用类 Python 的花括号语法，支持 # 注释。

basic.agent
let name = "Alice"
let count = 10
print("Hello, " + name)

# 输入输出
let q = input.prompt("Ask: ")
output.file("result.txt", q)

变量与类型

AFL 支持动态类型，自动类型推断，也支持显式类型注解。

基础类型

类型说明示例
string字符串"hello", 'world'
number数值（整数/浮点数）42, 3.14
boolean布尔值true, false
null空值null
list有序列表[1, 2, 3]
dict键值对{"key": "value"}

类型注解

类型注解示例
let name: string = "Alice"
let count: number = 10
let items: list = [1, 2, 3]
let config: dict = { timeout: 30, retry: true }

# 运行时类型检查
if type(config) == "dict" {
    let timeout = config.timeout
}

# 类型转换
let num = int("42")
let str = str(123)

控制流

if / else if / else

if.agent
if score >= 90 {
    print("A")
} else if score >= 80 {
    print("B")
} else {
    print("C")
}

循环

loop.agent
# for-range 循环
for i in range(5) {
    print(i)
}

# for-in 列表
for item in ["a", "b", "c"] {
    print(item)
}

# while 循环
let x = 5
while x > 0 {
    x = x - 1
}

# break / continue
for i in range(10) {
    if i >= 5 { break }
    if i % 2 == 0 { continue }
    print(i)
}

函数

function.agent
# 基本函数
function add(a, b) {
    return a + b
}

# 带返回类型声明
function greet(name) -> string {
    return "Hello, " + name + "!"
}

# 默认参数值
function connect(host, port=8080, timeout=30) {
    print(host + ":" + str(port))
}

# 关键字参数
connect("localhost", timeout=60)

# 递归
function factorial(n) {
    if n <= 1 { return 1 }
    return n * factorial(n - 1)
}

print(factorial(5))  # 120

错误处理

try-catch.agent
try {
    let resp = api.call("GET", "https://api.example.com")
    if resp.status == 200 {
        print("Success")
    } else {
        throw "HTTP " + resp.status
    }
} catch error {
    log.error("Request failed: " + error)
} finally {
    print("cleanup")
}

# 断言
assert add(2, 3) == 5
assert_eq(len(items), 3)

知识库 (kb)

Skill 技能系统 (skill)

内置预设技能库（math、text、data），开箱即用；支持从 .agent/.afl 文件导入。

方法说明
skill.load(name)导入 skill（预设名或文件路径）
skill.run(name, opts)运行 skill，opts = { action, ...params }
skill.list()列出已导入 skills
skill.<name>.<method>(...)点号直接调用 skill 方法
skill["<name>"].<method>(...)方括号访问 skill

math 预设 skill

异步与并发

async.agent
# 异步执行
async {
    let result = await llm.prompt("Hello")

}

# 并行执行
parallel {
    let a = api.call("GET", "https://api1.example.com")
    let b = api.call("GET", "https://api2.example.com")
}
wait(all)  # 等待所有完成

测试框架

test.agent
# 单个测试
test "should add correctly" {
    assert add(2, 3) == 5
}

# 测试套件
suite "Calculator" {
    test "addition" {
        assert add(2, 3) == 5
    }
    test "subtraction" {
        assert sub(5, 3) == 2
    }
}

# 运行测试
test.run()           # 运行所有
test.run("Calculator")  # 运行指定套件
提示: 测试支持 assert、assert_eq 断言。 测试失败时会抛出 AssertionError 并显示详细信息。
